# Prompt Vault

AI 이미지 생성 프롬프트를 관리하는 개인용 웹앱입니다.

## 주요 기능

- 프롬프트 저장/편집/복제/삭제
- 태그 기반 분류 (그룹 관리, 드래그 정렬)
- 프롬프트 조각(Fragment) — 자주 쓰는 문구 조합
- 참조 이미지 갤러리 (라이트박스 뷰어)
- 별점, 즐겨찾기, 메모
- JSON 내보내기/가져오기

## 기술 스택

- **프론트엔드:** 단일 HTML 파일
- **백엔드:** Google Apps Script + Google Sheets
- **암호화:** AES-256-GCM (클라이언트 사이드, Web Crypto API)
- **이미지 저장:** Google Drive (암호화 업로드)
- **호스팅:** GitHub Pages

## 보안

모든 데이터는 사용자 비밀번호 기반으로 브라우저에서 암호화된 후 저장됩니다. 서버(Google)에는 암호화된 데이터만 존재하며 비밀번호 없이는 내용을 확인할 수 없습니다.
